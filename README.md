# Labirent Oyunu
Bu proje, backtracking (geri izleme) algoritması ve stack veri yapısı kullanılarak metin dosyasından okunan labirent haritalarını çözmek için geliştirilmiştir.
## Technologies
- C++
- Backtracking Algorithm
- Stack Data Structure
- File I/O
## Features
- Metin dosyasından labirent okuma
- Backtracking ile yol bulma
- Stack kullanarak adım takibi
- Çözüm yolunun hesaplanması
## How to Run
1. Projeyi indir
2. C++ compiler ile derle
3. Programı çalıştır
4. Labirent dosyasını input olarak ver
